About This Dashboard

What is 287(g)?

Section 287(g) of the Immigration and Nationality Act allows ICE to deputize local law enforcement to perform immigration enforcement functions. There are three types:

  • Jail Enforcement Model: Local jails screen inmates for immigration status
  • Warrant Service Officer: Local officers can serve ICE warrants
  • Task Force Model: Local officers do immigration enforcement in the field

Data Sources

  • 287(g) Agreements: ICE 287(g) Program (updated weekly)
  • State Policies: ILRC State Map on Immigration Enforcement 2024

Limitations

This dashboard only shows formal 287(g) agreements. Many jurisdictions cooperate with ICE without formal agreements by:

  • Honoring ICE detainers (requests to hold people)
  • Notifying ICE of release dates
  • Allowing ICE access to jails
  • Sharing information informally

For county-level cooperation data beyond 287(g), see the ILRC Local Enforcement Map (note: last updated 2019).
