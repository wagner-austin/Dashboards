Metabolomics Analysis Plan

Drought Effects on Coastal Sage Scrub Metabolomics

Project Overview

Analysis of metabolomics data from coastal sage scrub plants under different water treatments. Focus on identifying compounds that are enhanced or suppressed by drought conditions in leaves and roots.

Data Scope: Using Emily's data only. Jaycee's data excluded from current analysis.

Key Variables

  • Organs: Leaves (L) vs Roots (R)
  • Treatments: Drought, Watered/Irrigated, Ambient
  • Sites: Loma Ridge, EcoPreserve
  • Species: Artemisia, Xeric, Mesic

Sample Assignments (Emily)

Data file: Emily_Data_Pruned_Labeled.xlsx

Leaves (L) - 12 samples

Treatment Samples n
Drought BL, CL, EL, GL 4
Ambient IL, JL, LL, ML 4
Watered OL, PL, RL, TL 4

Roots (R) - 11 samples

Treatment Samples n
Drought AR, DR, ER, GR 4

Analysis Pipeline

1. Data Cleaning

Done
  • Filter out blank peaks (peaks in blank but not sample)
  • Flag peaks: only in sample, only in blank, in both
  • Flags in Excel: Sample Only, Blank Only, Both >90%
Results:
Sample Only: 7,560 peaks (32.7%) - usable for analysis
Blank Only: 1,879 peaks (8.1%) - contamination, exclude
Both >90%: 5,720 peaks (24.7%) - ambiguous

3. Chemical Diversity (Shannon Index)

To Do
  • Calculate abundance (integrated values for all compounds)
  • Calculate proportion of each compound
  • Shannon Diversity: H = -Σ(p × ln(p))
  • Higher H = more diverse
  • Create sheet: abundance → proportion → p*ln(p) → Sum → H
