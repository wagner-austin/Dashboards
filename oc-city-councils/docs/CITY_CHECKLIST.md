## Priority Actions

### High Priority (affects data accuracy)

1. **Research term limits for all cities**
   - Check municipal code for each city
   - Document ordinance numbers and effective dates
   - Note if city has NO term limits (explicit null)

2. **Enrich election history from OC Registrar**
   - Run `enrich_yaml.py` for each city
   - Verify vote counts match official records
   - Add missing candidate lists

### Medium Priority (improves completeness)

3. **Verify current council members**
   - Check city websites for updates
   - Confirm term end dates

4. **Add missing document_center and municipal_code URLs**
   - Many cities have these in comments but not all

### Low Priority (nice to have)

5. **Add historical elections before 2016**
   - 2014 and 2012 data is available
   - Lower priority since older data

## How to Update This Checklist

After completing work on a city:

1. Run validation: `python election_data/validate_against_yaml.py`
2. Update the appropriate row in this checklist
3. Commit both the YAML and this checklist
